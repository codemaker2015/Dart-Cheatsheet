# Dart cheat sheet

This project starts from **Hello World** until the end of the **Dart** :)

Online compiler : [dartpad](https://dartpad.dev/)

### Simple example

- **Hello World** in **Dart**

```dart
void main() {
    // Just print it!
    print("Hello World");
}
```

```dart
enum Os { BigSur, Mojave, Sierra, Catalina }
```
